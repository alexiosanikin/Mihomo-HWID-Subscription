# Mihomo HWID Subscription Installer

Установщик подписок с требованием HWID для Mihomo для XKeen на роутерах Keenetic (KeeneticOS / Entware) и терминальных версиях Linux.

## Установка

Для чистой Entware-среды:
```sh
opkg update
opkg install curl
```

`XKeen`
```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/dorian6996/Mihomo-HWID-Subscription/main/install.sh)"
```

Если XKeen не установлен, скрипт поставит XKeen с Mihomo автоматически.

`Linux`
```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/dorian6996/Mihomo-HWID-Subscription/main/install-linux.sh)"
```

## Особенности

Скрипт может добавить/заменить подписку с перезаписью уже добавленных подписок.  
При запросе он отдаёт следующие заголовки:

- `x-hwid` — HWID, сгенерированный из MAC-адреса
- `x-device-os` — KeeneticOS
- `x-ver-os` — версия ОС (напр. `5.0.4`)
- `x-device-model` — модель роутера (напр. `GigaKN1012`)
- `User-Agent` — `mihomo/<версия ядра>`

Если подписка отдаёт заголовок `profile-title`, имя берётся из него.  
Иначе задаётся имя вида `subДДММГГ` (дата добавления).  
Если подписка с таким именем уже существует — добавляется суффикс `_2`, `_3` и т.д.

## Сценарии работы

**Если `config.yaml` уже существует**, скрипт предлагает:

1. **Добавить новую подписку** — добавит провайдер к уже существующим
2. **Заменить существующие подписки** — перезапишет все добавленные провайдеры
3. **Полностью заменить конфиг** — перезапишет конфиг целиком с новой подпиской

**Если `config.yaml` отсутствует** (или выбран вариант с полной заменой), предлагается создать:

1. **Полный конфиг** — с заполнением из `template.yaml` этого репозитория
2. **Минимальный конфиг** — один глобальный прокси, весь трафик идёт через него

При создании полного конфига будет предложено настроить **автообновление** раз в 3 дня в 5:00.
