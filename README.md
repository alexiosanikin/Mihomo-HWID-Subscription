 
# Mihomo HWID Subscription Installer

Установщик подписок с требованием HWID для Mihomo для XKeen на роутерах Keenetic (KeeneticOS / Entware).

## Установка

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/dorian6996/Mihomo-HWID-Subscription/main/install.sh)"
```

## Особенности 

Скрипт может добавить/заменить подписку с перезаписью уже добавленных подписок, он отдает хедеры, содержащие следующие параметры:
       "x-hwid: HWID сгенерированный из MAC" 
       "x-device-os: Keenetic OS"
       "x-ver-os: Версия ОС(напр:5.0.4)"
       "x-device-model: Модель Роутера(напр:GigaKN1012)"
       "User-Agent: mihomo/Установленная версия ядра mihomo"
 Если подписка отдает хедер profile-title, то имя подписки будет браться из него, иначе будет задано имя subДДММГГ, где ДДММГГ - дата добавления подписки, если подписка с таким именем существует, то к имени будет добавлен суффикс _2,_3,etc. 

Если config.yaml уже существует в папке etc/mihomo, скрипт предлагает:
1) Добавить новую подписку - Добавит подписку к уже существующей.
2) Заменить существующие подписки - Перезапишет все уже добавленные.
3) Полностью заменить конфиг - Перезапишет конфиг целиком с добавлением новой подписки.

Если config.yaml отсутствует или выбран вариант с заменой конфига, то будет предложено 2 варианта:
1) Создать полный конфиг с заполнением из файла template.yaml из этого репозитория.
2) Создать минимальный рабочий конфиг с 1 единственной группой, в которую будет идти весь трафик.

При замене или создании полного конфига будет предложено создание расписания для обновления конфига раз в 3 дня в 5:00.
